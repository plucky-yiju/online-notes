# Markdown 语法指南

本文介绍常用的Markdown语法，帮助你快速上手写作。

## 标题

使用 `#` 号表示标题，支持1-6级：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 文本样式

| 样式 | 语法 | 效果 |
|------|------|------|
| 粗体 | `**文本**` | **文本** |
| 斜体 | `*文本*` | *文本* |
| 删除线 | `~~文本~~` | ~~文本~~ |
| 行内代码 | `` `代码` `` | `代码` |

## 列表

### 无序列表

```markdown
- 项目一
- 项目二
- 项目三
```

效果：
- 项目一
- 项目二
- 项目三

### 有序列表

```markdown
1. 第一步
2. 第二步
3. 第三步
```

效果：
1. 第一步
2. 第二步
3. 第三步

## 链接和图片

```markdown
[链接文字](https://example.com)
![图片描述](image.jpg)
```

## 引用

```markdown
> 这是一段引用文字
> 可以有多行
```

效果：

> 这是一段引用文字
> 可以有多行

## 代码块

使用三个反引号包裹代码，并指定语言：

````markdown
```javascript
function hello() {
    console.log('Hello, World!');
}
```
````

效果：

```javascript
function hello() {
    console.log('Hello, World!');
}
```

## 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A1  | B1  | C1  |
| A2  | B2  | C2  |
```

效果：

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A1  | B1  | C1  |
| A2  | B2  | C2  |

## 分割线

使用三个或更多的 `-` 或 `*`：

```markdown
---
```

---

## 任务列表

```markdown
- [x] 已完成任务
- [ ] 未完成任务
```

效果：
- [x] 已完成任务
- [ ] 未完成任务

---

现在你已经掌握了基本的Markdown语法，开始写作吧！
